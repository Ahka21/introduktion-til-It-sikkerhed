Denne mappe indeholder dokumentation og filer, der understøtter den skriftlige rapport.
Bilagene bruges som dokumentation for det praktiske arbejde, der er beskrevet i rapporten.


Kali/

Indeholder screenshots fra arbejde i Kali Linux, herunder:
- nmap portscans
- Wireshark netværksanalyse

Disse bilag dokumenterer min praktiske erfaring med centrale værktøjer i Kali Linux,
som beskrevet i rapportens afsnit om Kali Linux og sikkerhedsværktøjer.


Terminal/

Indeholder screenshots, der viser brug af terminalen og kommandolinjen,
herunder oprettelse af mapper, filhåndtering og brug af kommandoer som ls og grep.

Dokumenterer min fortrolighed med terminalen og effektiv brug af kommandolinjen,
som beskrevet i rapportens afsnit om terminalarbejde.

--------------------------------
Bash/
--------------------------------
Indeholder Bash-scripts, der er udviklet til automatisering af simple opgaver,
samt screenshots af script-output.

Dokumenterer min evne til at udvikle og anvende Bash-scripts til automatisering
på Linux-systemer.

--------------------------------
Python/
--------------------------------
Indeholder Python-scripts samt screenshots af kørsel og output.

Dokumenterer mit arbejde med Python scripting, herunder brug af input, output
og grundlæggende programstruktur, som beskrevet i rapporten.

--------------------------------
Powershell/
--------------------------------
Indeholder simple Powershell-scripts til visning af systeminformation.

Dokumenterer min grundlæggende forståelse for Powershell og automatisering
på tværs af systemer.

--------------------------------
Kryptografi/
--------------------------------
Indeholder filer og screenshots relateret til kryptografi, herunder:
- hashing (SHA-256)
- symmetrisk kryptering (AES-256)
- oprettelse af selvsigneret certifikat

Dokumenterer min forståelse for grundlæggende kryptografiske begreber
og korrekt anvendelse af algoritmer, som beskrevet i rapportens kryptografi-afsnit.
